# Experiment 8.1

## Title  
Login Form with React State Management using React Hooks

---

## Description  
This project implements a simple login form using React.js with client-side state management. It demonstrates how to build interactive UI components using React hooks, handle form inputs, validate user data, and display dynamic feedback based on user actions.

The application uses controlled components to manage form state and performs basic validation before allowing login. This experiment focuses on React fundamentals like state handling, event handling, and conditional rendering.

---

## Learning Outcomes  
- Understand React functional components and hooks (useState)  
- Implement controlled form inputs in React  
- Perform client-side form validation  
- Manage UI state dynamically using React  
- Handle form submission events  
- Implement conditional rendering based on authentication state  

---

## Tech Stack  
- React 18+  
- JavaScript (ES6+)  
- HTML5  
- CSS3  
- Node.js (for React environment)

---

## Features  
- Responsive login form UI  
- Username and password validation  
- Real-time error messages  
- Conditional rendering (Login / Dashboard view)  
- Simple authentication logic (client-side demo)
